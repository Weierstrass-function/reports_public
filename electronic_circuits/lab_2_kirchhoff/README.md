# Универсальный алгоритм

1. Обозначить существующие токи.

<img width="261" height="207" alt="image" src="https://github.com/user-attachments/assets/5ebda9e8-e20c-4e78-9d1b-e64452975664" />

>Если направление тока неочевидно - наугад, если ошибемся после решения Системы Линейных Уравнений (далее ) будет отрицательное значение этого тока.

3. Составить уравнения по первому правилу:
```
I_1 - I_3 - I_4 = 0
I_2 + I_3 - I_5 = 0
I_4 + I_5 - I_6 = 0
```
>Первое правило: сумма входящих в узел равна сумме выходящих

3. Выбрать контуры:

<img width="654" height="553" alt="image" src="https://github.com/user-attachments/assets/12b81f52-8546-4490-841f-db2ca26240a3" />

>Задача проста - обойти все ребра (ветви) графа.

4. Составить уравнения ЭДС по второму правилу и закону Ома:
```
1*I_1 + 4*I_4 = 1
-1*I_1 + 2*I_2 + 5*I_5 - 4*I_4 = 0
1*I_1 - 2*I_2 + 3*I_3 = 0
```

>Второе правило: в любом замкнутом электрическом контуре алгебраическая сумма ЭДС равна алгебраической сумме падений напряжения на всех сопротивлениях этого контура

>U = R*I

>Направление обхода совпадает с направлением тока (из шага 1) +, иначе -

>Источник напряжения (ЭДС) есть - берем его ЭДС (например на иллюстрации 1 вольт), иначе 0

4. Если все сделано верно должна получится решаемая СЛУ
